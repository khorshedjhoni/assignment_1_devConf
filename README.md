# 🚀 DevConf 2026 – Developer Conference Landing Page

A modern, responsive one-page developer conference website built using **HTML5** and **CSS3**. This project was created as part of a web development assignment to practice semantic HTML, Flexbox, CSS Grid, and responsive web design.

## 🌐 Live Demo

👉 https://khorshedjhoni.github.io/assignment_1_devConf/

---

## 📌 Features

- Responsive navigation bar
- Hero section with background image and dark overlay
- Meet the Speakers section with speaker cards
- Pricing section with three plans
- Highlighted featured pricing plan (Pro)
- AI-generated "Why Join Us at DevConf 2026?" section
- Modern footer with social media icons
- Responsive layout using CSS Grid and Flexbox

---

## 🛠️ Built With
